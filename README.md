## http & stream
  - `git clone https://github.com/yutin1987/uf.git`
  - `npm install`
  
### Run

  - `npm run http`
  - `npm run local`

OR

  - `npx ts-node index.ts <url|path>`
  
### Outfile
  `./${Date.now()}.jpeg`

### Testing
  `npm run test`
